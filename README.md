mc-stan.vim
===========

#### Current state

---

This plug-in is usable but currently under heavy development.

#### Features

---

- Syntax matching for the stan modeling language

- Snippets using the [UltiSnips](http://github.com/SirVer/ultisnips) engine.

- Indentation using cindent
